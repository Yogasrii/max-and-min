# Maximum and Minimum Number Finder in Python

This repository contains a simple Python program that finds the maximum and minimum values from a list of numbers.

## Description

The program uses Python's built-in `max()` and `min()` functions to determine the largest and smallest numbers in a given list.

## Features

* Finds the maximum value in a list
* Finds the minimum value in a list
* Simple and beginner-friendly code
* Useful for learning Python basics

## Code

```python
numbers = [10, 25, 5, 40, 18]

print("Maximum:", max(numbers))
print("Minimum:", min(numbers))
```

## Sample Output

```text
Maximum: 40
Minimum: 5
```

## How It Works

1. Create a list of numbers.
2. Use `max()` to find the largest value.
3. Use `min()` to find the smallest value.
4. Display the results.
